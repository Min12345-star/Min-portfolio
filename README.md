# MIN XIE Marketing Portfolio

Static portfolio website for GitHub Pages deployment.

## Files

- `index.html`: interactive folder homepage
- `about.html`: profile, education, awards, GPA, and skills
- `experience.html`: internship and project experience timeline
- `projects.html`: selected project portfolio
- `contact.html`: email and phone
- `styles.css`: full visual system and responsive layout
- `script.js`: folder hover interaction
- `assets/`: project images and PDF

## Customize

Replace the placeholder text in `about.html` and `experience.html` with your final school, awards, GPA, internship companies, and dates.

To add your portrait, place an image in `assets/`, then replace the `.portrait-placeholder` block in `about.html` with:

```html
<img class="portrait-image" src="assets/your-photo.jpg" alt="Portrait of Min Xie">
```

## Deploy

Upload all files in this folder to a GitHub repository. For GitHub Pages, set Pages source to the repository root and open the generated Pages URL.
